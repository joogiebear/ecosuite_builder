§ I

Built for people who actually run servers.

01

Schema-aware editors

Every plugin knows its own fields, types, and defaults. Hover any input for the docs blurb. Wrong type? You can’t enter it.

02

Live YAML preview

See the file you’re actually shipping — updated as you type, with the same indentation EcoSuite expects.

03

Reusable library

Save effects, conditions, triggers, and whole entries to a personal library. Drop them into any plugin in two clicks.

04

Pack import

Drag in an existing config pack and the studio reverse-engineers it — no clean-room rebuilds.

05

EcoMenus designer

Lay out menus with a visual slot grid. Configure pages, masks, fillers, and click handlers without counting characters.

06

Cream & coal themes

Light paper for day work, deep ink for late builds. Both theme-aware, both readable for hours.
